## AppBadge

Example:

<Source
  code={`
    <app-badge
        :label='label'
        :is-background-transparent="isBackgroundTransparent"
        :is-borderless="isBorderless"
        :status='status'
    />`}
/>

We have defined statuses in constant called **BADGE_STATUSES** with all have own colors and borders. Each status have
own label so you don't need write your own.
