## AppActionGroup

Example:

<Source code={`<app-action-group :config="config" :rows="rows" />`}></Source>

AppActionGroup component is for configurable button group. Accepts rows and config.

## Group config example

```js
[
  {
    title: 'action.download_receipt',
    icon: 'icon-transfer-from',
    disabled: false,
    event: 'onTransfer',
  },
  {
    title: 'action.make_recurrent',
    icon: 'icon-refresh',
    disabled: false,
    event: 'onTransfer',
  },
];
```
