## AppWalletCard

Example:
<Source code={
  `<app-wallet-card
        title="Money in"
        subtitle="USD"
        prefix-symbol="$"
        money-flow="in"
        value="2500"/>
`}/>

This `wallet card` has predefined image with connected to prop `money-flow` and shows arrow which can be in 2
conditions:
 `in` (money-flow="in" - arrow up) and `out` (arrow down). Without `money-flow` prop, you should set prop `img-url`
 to set image on right top corner.
